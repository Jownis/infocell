    •          1 Iniciar o Repositório Git (Inicie o Git dentro dessa pasta)
cd c/progSistemas/seu-nome/loja-infocell

    git init
2 Adicione uma nova ramificaçção (brench) chamada pagina inicial
cd c/progSistemas/seu-nome/loja-infocell/public

git checkout -b pagina-inicial (nome-da-brench)

3 Adicione os 2 arquivos vazios index.html e favicon.ico dentro da pasta public
touch index.html favicon.ico


 4 Verificar Status de alterações no repositório
    git status

5 Adicionar Arquivos ao Repositório Local ( adicionar todos os arquivos criados ao seu repositório Git local)

    git add .

 6 Fazer o Primeiro Commit  (Após adicionar os arquivos, faça o commit para salvar as alterações)

    git commit -m "Inicializando o projeto loja-infocell"

 7 Criar o Repositório no GitHub  (Abra o GitHub e crie um novo repositório)

    No GitHub, clique em New Repository.
    Dê o nome de loja-infocell ao repositório.
    Marque as opções desejadas (pode deixar marcado o README.md).
    Clique em Create repository.

8 Conectar o Repositório Local ao Repositório no GitHub

No Git Bash, execute o comando para conectar seu repositório local ao GitHub:

    git remote add origin https://github.com/seu-usuario/loja-infocell.git

9 Enviar o Projeto para o GitHub ( Envie seu código para o GitHub)

    git push -u origin (nome-da-branch)

---------------------------------------------------------------------------------------------


Se
você deseja alterar a URL do repositório remoto, use o comando abaixo,
substituindo URL_DO_REPOSITORIO pela nova URL do repositório:

git remote set-url origin URL_DO_REPOSITORIO



Após alterar, verifique se a URL foi alterada com:

git remote -v
